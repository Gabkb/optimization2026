Este repositório será utilizado para disponibilização de materiais, atividades e submissão dos trabalhos pelos alunos.

---

## 📁 Estrutura do Repositório

```
optimization2026/
├── materiais/
├── atividades/
└── alunos/
```

### 🔹 `materiais/`
Contém os materiais de cada aula.

```
materiais/AULAXX/
```

Exemplos:
- Slides
- Notas de aula
- Códigos (Julia / Python)
- Referências

---

### 🔹 `atividades/`
Contém os enunciados e arquivos base das atividades.

```
atividades/AULAXX/
```

Exemplos:
- PDFs com enunciado
- Notebooks (.ipynb)
- Bases de dados
- Scripts auxiliares

---

### 🔹 `alunos/`
Cada aluno deve criar sua própria pasta utilizando seu **username do GitHub**:

```
alunos/<github_username>/
```

Dentro dela, organizar as submissões por aula:

```
alunos/<github_username>/AULAXX/
```

## 📏 Convenções

- As aulas devem seguir o padrão:

  ```
  AULA01, AULA02, ..., AULA10, ...
  ```

- O nome da pasta do aluno deve ser exatamente o **username do GitHub**

- Cada atividade deve ser submetida dentro da pasta da aula correspondente

---

## 📥 Submissão de Atividades

Cada aluno deve:

1. Criar sua pasta em:
   ```
   alunos/<seu_username>/
   ```

2. Criar a estrutura por aula:
   ```
   alunos/<seu_username>/AULAXX/
   ```

3. Inserir os arquivos da atividade dentro da pasta correspondente

4. Coloque no README.md da sua pasta o seu nome completo e número de matrícula para alunos regulares. Alunos especiais somente nome completo.

---

## ⚠️ Observações Importantes

- Não alterar arquivos de outros alunos
- Não modificar a estrutura do repositório
- Manter organização e nomenclatura padrão
- Submeter apenas arquivos necessários
